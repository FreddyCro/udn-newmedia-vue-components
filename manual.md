## 使用方式
> 1. 新建vue spa project(vue-cli)
> 2. 直接使用公版修改(webpack)

### Project setup
```
npm install
```
### Compiles and hot-reloads for development
```
npm run dev
```
### Compiles and minifies for production
```
npm run build
```
